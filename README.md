# Custom HTML & CSS Error Pages

- SVG Backgrounds
- Animated Aspecs
- CSS Config

## Included Pages:

- 400 for Bad Requests.
- 401 for Unauthorized Access.
- 403 for No Access.
- 404 for Page Not Found.
